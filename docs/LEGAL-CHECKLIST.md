# CampoTech Legal Review Checklist

This checklist ensures compliance with Argentine laws and regulations.

## Ley 25.326 - Protección de Datos Personales

### Privacy Policy Requirements
- [ ] Privacy policy exists and is accessible
- [ ] Policy is written in Spanish
- [ ] Policy is easy to understand (plain language)
- [ ] Policy includes:
  - [ ] Data controller identification (CampoTech S.A.)
  - [ ] Purpose of data collection
  - [ ] Types of data collected
  - [ ] Legal basis for processing
  - [ ] Data retention periods
  - [ ] Third-party sharing (MercadoPago, AFIP, etc.)
  - [ ] User rights (access, rectification, deletion)
  - [ ] Contact information for data requests
  - [ ] Security measures description
  - [ ] Cookie policy

### Data Subject Rights
- [ ] Right to access personal data implemented
- [ ] Right to rectify data implemented
- [ ] Right to delete data implemented (with legal exceptions)
- [ ] Process for data requests documented
- [ ] 10-day response timeline documented

### Data Collection Consent
- [ ] Explicit consent obtained at signup
- [ ] Consent checkbox is not pre-checked
- [ ] Separate consent for marketing communications
- [ ] Consent is recorded and timestamped
- [ ] Consent can be withdrawn

### Data Security Measures
- [ ] Data encryption at rest
- [ ] Data encryption in transit (HTTPS)
- [ ] Access controls implemented
- [ ] Audit logging enabled
- [ ] Breach notification procedure documented

### Data Retention Policy
- [ ] Retention periods defined:
  - [ ] User account data: Duration of account + 5 years
  - [ ] Verification documents: 10 years (AFIP requirement)
  - [ ] Payment records: 10 years (tax requirement)
  - [ ] Employment records: 10 years (Ley 20.744)
  - [ ] Audit logs: 5 years
- [ ] Deletion procedures documented
- [ ] Anonymization procedures for analytics

## Ley 24.240 - Defensa del Consumidor

### Consumer Rights
- [ ] 10-day cooling-off period implemented
- [ ] Full refund during cooling-off period
- [ ] Clear cancellation process
- [ ] No penalty for cancellation within 10 days

### Price Transparency
- [ ] All prices include IVA
- [ ] Final price shown before payment
- [ ] No hidden fees
- [ ] Currency clearly stated (ARS)

### Service Terms
- [ ] Terms of service exist and are accessible
- [ ] Terms are written in Spanish
- [ ] Terms include:
  - [ ] Service description
  - [ ] Price and billing terms
  - [ ] Cancellation policy
  - [ ] Refund policy
  - [ ] Service level commitments
  - [ ] Limitation of liability
  - [ ] Dispute resolution

### Cancellation Policy
- [ ] Cancellation process is clear
- [ ] User can cancel online
- [ ] No excessive cancellation barriers
- [ ] Refund timeline stated (72 hours for Ley 24.240)
- [ ] Pro-rata refund policy documented

### Support Requirements
- [ ] Contact information displayed
- [ ] Support available in Spanish
- [ ] Response time commitments stated
- [ ] Complaint handling procedure documented

## AFIP Compliance

### Tax Documentation
- [ ] CUIT validation implemented
- [ ] IVA condition captured
- [ ] Invoice generation compliant
- [ ] Invoice types (A, B, C) correctly assigned
- [ ] Electronic invoice integration (if applicable)

### Data Submission
- [ ] Required AFIP data collected
- [ ] Data format compliant with AFIP requirements
- [ ] Reporting timelines understood

### Record Keeping
- [ ] 10-year retention for tax documents
- [ ] Audit trail for all transactions
- [ ] Records accessible for inspection

## Employment Law (Ley 20.744)

### Employee Verification
- [ ] Proper employee identification collected
- [ ] Employment relationship documentation
- [ ] 10-year record retention

### Contractor vs Employee
- [ ] Clear contractor relationship terms
- [ ] Contractor documentation requirements
- [ ] Compliance with contractor regulations

✓ Entiendo que mi suscripción se renovará automáticamente cada mes
✓ Puedo cancelar en cualquier momento desde la sección de Facturación
