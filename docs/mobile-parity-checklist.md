This document tracks feature parity between the CampoTech mobile app and web dashboard. Our core principle: **A plumber starting their business with only a smartphone must be able to run their entire operation from the CampoTech mobile app.**

## Priority Legend

- **P0** - Critical for MVP launch
- **P1** - Important for good user experience
- **P2** - Nice to have, can be web-only

## Web-Enhanced Features

These features are intentionally more powerful on web due to screen size and input method:
- Bulk operations (scheduling, invoicing)
- Detailed reports with exports
- Drag-and-drop interactions
- Data export (CSV, Excel)

## Mobile-Enhanced Features

These features are unique or better on mobile:
- Voice input for job creation
- GPS navigation integration
- Camera for photos/signatures
- One-tap calling
- One-tap WhatsApp
- Background location tracking
- Push notifications
- Offline operation

## Testing Checklist

Before any release, verify:

- [ ] New user can complete full signup on mobile only
- [ ] User can manage entire team from mobile
- [ ] User can create, assign, and complete jobs on mobile
- [ ] User can create and send invoices from mobile
- [ ] User can view basic reports on mobile
- [ ] All features work offline with proper sync
- [ ] App cold start < 4 seconds
- [ ] RAM usage < 150MB
- [ ] No features require web dashboard access
