## Resumen Ejecutivo

**CampoTech** es una plataforma integral de gestión de servicios de campo diseñada específicamente para el mercado argentino. Nuestra solución permite a empresas de servicios (plomeros, electricistas, técnicos de aire acondicionado, gasistas, cerrajeros, pintores, etc.) gestionar todo su negocio desde una única plataforma.

### Propuesta de Valor

```
┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│   "Organizá tu negocio de servicios como un profesional"               │
│                                                                         │
│   ✓ Gestioná trabajos y técnicos                                       │
│   ✓ Facturá con AFIP integrado                                         │
│   ✓ Atendé clientes por WhatsApp con IA                                │
│   ✓ Controlá inventario y vehículos                                    │
│   ✓ Aparecé en nuestro marketplace de consumidores                     │
│                                                                         │
│   Todo desde tu celular o computadora.                                  │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### Cifras Clave

| Métrica | Objetivo |
|---------|----------|
| Mercado objetivo | Empresas de servicios en Argentina |
| Precio mensual | $25 - $120 USD |
| App consumidores | GRATIS para usuarios finales |
| Diferenciador | WhatsApp AI + Marketplace integrado |

## El Problema que Resolvemos

### La Realidad de los Trabajadores de Servicios en Argentina

```
Juan es plomero. Así maneja su negocio hoy:

📱 Recibe llamadas y WhatsApps todo el día
📝 Anota trabajos en un cuaderno (o se olvida)
🧾 Hace facturas a mano o en Excel
📦 No sabe qué materiales tiene en la camioneta
💸 Cobra en efectivo y pierde el control
📊 No tiene idea si gana o pierde plata
🔍 Consigue clientes por boca en boca solamente

Resultado: Desorganización, pérdida de clientes,
           ingresos impredecibles
```

### Los Problemas Específicos

| Problema | Impacto |
|----------|---------|
| **Gestión manual** | Olvida trabajos, llega tarde, pierde clientes |
| **Sin facturación digital** | Problemas con AFIP, no puede trabajar con empresas |
| **WhatsApp caótico** | Mensajes perdidos, clientes sin respuesta |
| **Sin control de inventario** | Compra materiales de más o le faltan en el trabajo |
| **Adquisición de clientes** | Depende 100% del boca en boca |
| **Sin métricas** | No sabe qué servicios son rentables |

### El Problema del Consumidor

```
María necesita un plomero urgente:

1. Pregunta a amigos/familia: "¿Conocés un buen plomero?"
2. Busca en Google: Encuentra páginas desactualizadas
3. Prueba MercadoLibre Servicios: Cobran comisión, poca confianza
4. Grupos de WhatsApp/Facebook: Caótico, sin garantías

Resultado: Incertidumbre, riesgo de estafas,
           sin forma de verificar calidad
```

**Fases de Crecimiento:**

1. **Fase 1 - Motor de Crecimiento**: Importar 61,000 matrículas públicas (ERSEP, CACAAV, Gasnor)
2. **Fase 2 - Activación**: Profesionales reclaman su perfil, inician trial de 21 días
3. **Fase 3 - Conversión**: ~15% convierte a suscripción paga después del trial
4. **Fase 4 - Efecto Red**: Más profesionales = mejor marketplace = más consumidores
5. **Resultado**: Crecimiento exponencial con CAC = $0

### Por Qué Funciona

| Competidor | Modelo | Problema |
|------------|--------|----------|
| MercadoLibre Servicios | Comisión por lead | Empresas evitan la plataforma |
| Apps de delivery | Comisión por transacción | Margen muy bajo para servicios |
| **CampoTech** | Suscripción fija + Trial gratis | Prueba sin riesgo, valor inmediato |

**Ventaja CampoTech**: Las empresas prueban la facturación AFIP por 21 días. Cuando ven el ahorro de tiempo, pagan.

## Los Usuarios del Sistema

### 1. CampoTech (Nosotros)

**Rol**: Dueños de la plataforma

**Acceso**: Panel de administración interno (`apps/admin`)

**Funciones**:
- Ver todas las empresas suscritas
- Gestionar pagos y suscripciones
- Entrenar la IA con conversaciones
- Ver métricas de crecimiento
- Soporte a empresas

---

### 2. Dueños de Empresas (Owner)

**Rol**: Propietarios de empresas de servicios

**Pago**: Suscripción mensual ($25 - $120 USD)

**Acceso**: Dashboard web completo + App móvil

**Funciones**:
| Área | Permisos |
|------|----------|
| Trabajos | Crear, asignar, ver todos |
| Clientes | Ver todos, editar, eliminar |
| Equipo | Agregar/eliminar usuarios, asignar roles |
| Facturación | Crear facturas, ver pagos, configurar AFIP |
| Inventario | Control total |
| Analytics | Todas las métricas + posición de mercado |
| WhatsApp | Configurar IA, ver conversaciones |
| Configuración | Marca, plantillas, notificaciones, suscripción |

---

### 3. Administradores (ADMIN)

**Rol**: Personal de oficina que coordina trabajos

**Pago**: Incluido en la suscripción del dueño

**Acceso**: Dashboard web (limitado) + App móvil

**Funciones**:
| Área | Permisos |
|------|----------|
| Trabajos | Crear, asignar, ver todos |
| Clientes | Ver todos, crear nuevos, editar |
| Equipo | Ver técnicos, NO puede agregar/eliminar |
| Facturación | ❌ SIN ACCESO |
| Inventario | Ver stock, solicitar reposición |
| Analytics | Métricas operativas (NO financieras) |
| WhatsApp | Ver y responder conversaciones |
| Configuración | ❌ SIN ACCESO |

---

### 4. Técnicos (Technician)

**Rol**: Trabajadores de campo que ejecutan los servicios

**Pago**: Incluido en la suscripción del dueño

**Acceso**: App móvil únicamente

**Funciones**:
| Área | Permisos |
|------|----------|
| Trabajos | Ver SUS trabajos asignados únicamente |
| Estado | Actualizar progreso (en camino, llegué, trabajando, terminado) |
| Reportes | Crear reportes por voz (dictado → texto automático) |
| Inventario | Ver stock de SU vehículo, registrar uso de materiales |
| Fotos | Tomar fotos antes/durante/después del trabajo |
| Firma | Capturar firma del cliente |
| Navegación | GPS al domicilio del cliente |
| Contacto | Llamar o WhatsApp al cliente |

---

### 5. Consumidores

**Rol**: Personas que buscan servicios

**Pago**: **GRATIS - Nunca pagan nada**

**Acceso**: App de marketplace

**Funciones**:
| Función | Descripción |
|---------|-------------|
| Buscar | Por categoría, ubicación, disponibilidad |
| Descubrir | Recomendaciones de IA basadas en necesidad |
| Ver perfiles | Información de empresas, fotos, calificaciones |
| Contactar | Abrir WhatsApp directamente con la empresa |
| Seguimiento | Ver progreso del trabajo contratado |
| Calificar | Dejar reseña después del servicio |

**Importante**: Los consumidores NO crean cuenta obligatoria. Simplemente usan la app para encontrar y contactar empresas.

## Funcionalidades Detalladas

### Gestión de Trabajos

```
CICLO DE VIDA DE UN TRABAJO

1. CREACIÓN
   └── Desde dashboard web
   └── Desde WhatsApp (IA crea automáticamente)
   └── Datos: cliente, dirección, servicio, fecha/hora

2. ASIGNACIÓN
   └── Dueño/Administrador asigna técnico
   └── Técnico recibe notificación push
   └── Trabajo aparece en su app

3. EJECUCIÓN
   └── Técnico actualiza estado:
       └── "En camino" → Cliente recibe ETA
       └── "Llegué" → Timestamp registrado
       └── "Trabajando" → Cliente ve progreso
       └── "Terminado" → Activa flujo de pago

4. DOCUMENTACIÓN
   └── Reporte de trabajo (voz o texto)
   └── Materiales usados (descuenta inventario)
   └── Fotos antes/después
   └── Firma del cliente

5. COBRO
   └── Efectivo: Técnico marca pagado (GPS + timestamp)
   └── MercadoPago: Link de pago automático
   └── Tarjeta: Técnico confirma pago
```

## El Marketplace de Consumidores

### Cómo Funciona para el Consumidor

```
1. ABRIR APP
   └── Auto-detecta ubicación
   └── Muestra categorías populares

2. BUSCAR
   └── Seleccionar categoría
   └── O describir problema (texto/voz)
   └── IA muestra recomendaciones

3. ELEGIR EMPRESA
   └── Ver tarjetas de empresas:
       ├── Nombre y logo
       ├── Calificación ⭐ (promedio)
       ├── Distancia 📍
       ├── Disponibilidad
       └── "Responde en X minutos"

4. VER PERFIL
   └── Información completa
   └── Fotos de trabajos
   └── Reseñas verificadas
   └── Lista de servicios

5. CONTACTAR
   └── Botón "WhatsApp" abre chat directo
   └── SIN intermediarios
   └── SIN comisiones

6. DESPUÉS DEL SERVICIO
   └── Recibe link de calificación
   └── Deja reseña
   └── Ayuda a otros consumidores
```

### Cómo Funciona para la Empresa

```
VISIBILIDAD EN MARKETPLACE

La empresa aparece automáticamente cuando:
✓ Tiene suscripción activa (cualquier plan)
✓ Perfil completo (nombre, servicios, fotos)
✓ Ubicación configurada

RANKING DE EMPRESAS

IA ordena empresas basándose en:
├── Distancia al consumidor (40%)
├── Disponibilidad actual (25%)
├── Calificación promedio (20%)
├── Tiempo de respuesta (15%)

Nota: NO hay ventaja por pagar más.
Todos los planes tienen misma visibilidad.
```

### Por Qué Es GRATIS para Consumidores

```
MODELO TRADICIONAL (competidores):
Consumidor busca → Plataforma cobra comisión → Empresa paga por lead
Resultado: Empresas evitan la plataforma, pocos proveedores

MODELO CAMPOTECH:
Consumidor busca → Contacta directo por WhatsApp → Empresa ya paga suscripción
Resultado: Más empresas participan, mejor servicio para consumidores

La plataforma gana con suscripciones, no con comisiones.
Por eso el marketplace puede ser 100% gratis.
```
