# New User Journey

## Flow Overview
This flow describes the path of a new user from landing page to their first successful action.

### Steps
1. **Acquisition**
    - Entry: [[Landing Page]]
    - Action: User clicks "Start Trial"
    - Next: [[Signup Page]]

2. **Onboarding**
    - User completes registration -> [[Verification Flow]]
    - User sets up profile -> [[App Onboarding]] (Wizard)

3. **Activation**
    - User lands on [[Dashboard Home]]
    - User creates first job -> [[New Job Page]]
